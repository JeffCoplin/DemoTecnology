# 📄 PRD — Technology Store RD Demo

## 1. 📌 Overview

Este proyecto consiste en desarrollar una **landing page moderna + catálogo interactivo** para una tienda de tecnología llamada **Technology Store RD**, con el objetivo de demostrar cómo pueden aumentar sus ventas utilizando una presencia digital optimizada.

La demo estará enfocada en:

* Captar atención (landing moderna)
* Mostrar productos (catálogo)
* Convertir clientes (WhatsApp)

---

## 2. 🎯 Objetivos del Producto

### Objetivo Principal

Crear una demo visualmente atractiva y funcional que permita a la tienda:

* Mostrar sus productos de forma profesional
* Generar confianza en clientes potenciales
* Convertir visitas en ventas a través de WhatsApp

### Objetivos Secundarios

* Demostrar autoridad como desarrollador
* Conseguir el cliente (venta del servicio)
* Mostrar automatización de ventas básica

---

## 3. 👥 Público Objetivo

* Personas interesadas en comprar tecnología en República Dominicana
* Clientes que buscan:

  * Buenos precios
  * Entrega rápida
  * Atención por WhatsApp

---

## 4. 🧩 Funcionalidades Principales

### 4.1 Landing Page

* Hero con mensaje claro de valor
* Botones de acción (Ver catálogo / WhatsApp)
* Secciones modernas con animaciones

### 4.2 Catálogo de Productos

* Grid responsive de productos
* Cada producto incluye:

  * Imagen
  * Nombre
  * Precio
  * Botón de compra (WhatsApp)

### 4.3 Integración con WhatsApp

* Botón flotante global
* Botón en cada producto
* Mensaje prellenado:

  > "Hola, quiero este producto: [Nombre]"

### 4.4 Carrito (UI Demo)

* Agregar productos
* Contador en navbar
* Vista básica (modal o dropdown)

### 4.5 Navegación

Navbar con:

* Inicio
* Catálogo
* Contáctanos
* Carrito
